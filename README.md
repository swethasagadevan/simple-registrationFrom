# 📝 Registration Form

A simple and responsive registration form built using **HTML**, **Tailwind CSS**, **CSS**, and **JavaScript**.

- Fully responsive layout
- Styled with Tailwind CSS and custom CSS

- Clean and modern UI
- Easy to integrate into any project
